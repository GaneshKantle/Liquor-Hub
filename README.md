# LiquorHub

LiquorHub is a J2EE-based e-commerce web application for licensed liquor retailers.

## Tech Stack

- Java
- JSP
- Servlets
- JDBC
- MySQL
- Apache Tomcat

## Features

- User Registration/Login
- Product Catalogue
- Shopping Cart
- Order Management
- Payment Module

## Project Structure

- DTO
- DAO
- Servlets
- JSP
- Utility Classes
